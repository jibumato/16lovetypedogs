🐶 MBTIタイプ別
「恋愛あるある」まとめ

自分のタイプを見たら「わかる！！」の嵐。友達に送りたくなる全16タイプ解説

好きな人ができたとき、デート中、告白するとき——MBTIのタイプによって行動も感情もこんなに違う。自分のタイプを読んで「わかる！！」ってなったらぜひ友達にも送ってみて。

🧠 NT（論理派）グループのあるある

INTJ
チワワ（建築家）
本命にだけ甘える一途な策士
  • 好きな人ができると、相手のことを無意識に「分析」しはじめる
  • 「この人は恋愛対象か？」と脳内で審査してから好きになるので、好意を持つまでが遅い
  • いざ好きになると、水面下でこっそり深く考えすぎて動けなくなる
  • 告白は完全にタイミングを計算してから実行する
  • 付き合い始めると「え、こんなに甘えてくるんだ」と相手にびっくりされがち
  • 嫉妬しても絶対に口には出さない。でも態度にちょっとだけ出る

→ INTJの恋愛詳細はこちら

INTP
ミニチュアシュナウザー（論理学者）
沼ると一途な恋の探究者
  • 「これって恋なの？単なる知的興味では？」と自問自答し続けて結論が出ない
  • 好きな人の発言を何時間でも反芻して意味を解析してしまう
  • 告白するタイミングを「最適化」しようとして永遠に動けない
  • 一度沼ると冷静なのにめちゃくちゃ一途になる
  • LINEの返信を考えすぎて2時間後に返す
  • 気づいたら相手の趣味に詳しくなっている（リサーチしてしまう）

→ INTPの恋愛詳細はこちら

ENTJ
コーギー（指揮官）
全力でリードする情熱の指揮官
  • 好きになったらアプローチが速すぎて相手がびっくりする
  • デートの計画を立てすぎてスケジュールが秒で埋まる
  • 「付き合う」と決めたら相手が返事する前に次の計画を考えている
  • 感情より論理で話すせいで「冷たい」と思われることがある
  • 相手の成長を本気で応援しすぎてコーチみたいになる
  • 別れた後も「なぜうまくいかなかったのか」を分析して反省レポートを作る（心の中で）

→ ENTJの恋愛詳細はこちら

ENTP
ジャックラッセル・テリア（討論者）
退屈が嫌いな刺激求める挑発屋
  • 好きな人に「それって本当？」と議論をふっかけてみる（好意の表れ）
  • 気を引こうとして少しいたずらな発言をしてしまい、後悔する
  • 「面白い人」に出会うと恋愛と知的興味の区別がつかなくなる
  • 同じ話を何度もするのが嫌で毎回新ネタを仕込んで会いに行く
  • 感情的になる相手に「なぜ？」と聞いてしまい空気を読めないと言われる
  • 飽き性と言われるけど、本当に好きな人には異常に執着する

💖 NF（感性派）グループのあるある

INFJ
キャバリア（提唱者）
静かに深く愛する孤高の共感者
  • 相手の感情を自分のことのように感じすぎて消耗する
  • 「この人のことが気になる」と感じた瞬間、その人の未来まで想像してしまう
  • 好きな人が悲しんでいると、理由も聞かずに寄り添ってあげたくなる
  • 心の壁が高すぎて、近づいてきた人を「試しているのかな」と疑ってしまう
  • 付き合っても「この人は本当に私を理解してる？」と不安になりがち
  • 深い会話ができる相手にだけ、急速に心を開いてしまう

→ INFJの恋愛詳細はこちら

INFP
マルチーズ（仲介者）
夢みる恋に全部賭ける理想主義者
  • 好きな人ができると、その人との妄想ストーリーが脳内で展開しはじめる
  • 「好き」と言えなくて、さりげないプレゼントや手紙で伝えようとする
  • ちょっと冷たくされただけで「嫌われた？」と一晩中悩む
  • 付き合うと「こんな素敵な人が私のそばにいていいの？」と幸福すぎてドキドキする
  • ケンカになると言葉が出てこなくて、あとで手紙を書いてしまう
  • 音楽・映画・本で「これ、まるで私の恋愛じゃん」と泣く
「わかる！！」ってなったらXでシェアしてね 🐾

→ INFPの恋愛詳細はこちら

ENFJ
ゴールデン・レトリバー（主人公）
みんなを幸せにしたい太陽系恋人
  • 好きな人の悩みを聞くために夜中まで付き合ってしまう（そして眠れなくなる）
  • 「あなたのために何かしたい！」という気持ちが前に出すぎて重いと思われることがある
  • グループの場でも好きな人のことを一番気にかけてしまう（バレてる）
  • 相手の感情を先読みしすぎて、本人より先に「疲れてる？」と察してしまう
  • 自分の気持ちより相手を優先しすぎて、自分が限界になるまで気づかない
  • 付き合うと記念日・誕生日・二人の思い出全部完璧に覚えている

→ ENFJの恋愛詳細はこちら

ENFP
トイ・プードル（運動家）
全力で恋するキラキラ自由人
  • 好きな人ができたら全エネルギーをその人に注ぎすぎて友達に心配される
  • 「付き合いたい！」という気持ちが高まりすぎて、まだ付き合ってないのに彼氏/彼女気分になる
  • 気持ちが盛り上がりすぎて先走った行動をして、後で「はやすぎた…」と反省する
  • デートは毎回「新しい体験」を提案したくて計画が過密になる
  • 好きな人に依存しすぎて「もっと自由にしてよ」と言われることがある
  • 別れた後も「あの人のこと、まだ好きかも」を引きずりやすい

🏡 SJ（誠実派）グループのあるある

ISTJ
柴犬（管理者）
一度決めたら一生の恋愛職人
  • 好意があっても顔や言葉に出せないので「脈なし」と思われがち
  • 「好きです」が言えなくて、代わりに傘を貸したり荷物を持ったりする
  • 記念日は必ずカレンダーに書いて絶対に忘れない
  • ケンカすると感情的になれず、黙り込んで相手をさらに不安にさせてしまう
  • 一度「この人」と決めたら、10年後も変わらず同じ温度で好きでいられる
  • 「もっと甘えてほしい」と言われても甘え方が分からなくて困る

→ ISTJの恋愛詳細はこちら

ISFJ
シーズー（擁護者）
気配り満点の縁の下の恋人
  • 好きな人の「好きな食べ物」「嫌いなもの」「体調」を全部把握している
  • 相手が何も言わなくても「疲れてそうだな」と察して先回りして動く
  • 「気を遣わないで！」と言われても、気を遣わないことができない
  • ケンカすると「自分が悪かったかも」と思って先に謝ってしまう（本当は相手が悪くても）
  • 「好き」という言葉より行動で愛情を伝えるので、伝わっていないことがある
  • 好きな人のために手作りお菓子を作ってしまう（渡せるかは別問題）

→ ISFJの恋愛詳細はこちら

ESTJ
ボストン・テリア（幹部）
頼れるリーダー気質の安定恋人
  • デートプランが具体的すぎて「旅行の計画書みたい」と言われる
  • 「もっと気楽にして」と言われても、気楽にすることへの不安がある
  • 好きな人に対してもズバッとアドバイスをしてしまい「きつい」と思われることがある
  • 相手を支えたい気持ちが強すぎて、いつの間にかマネージャーみたいになっている
  • 感謝の言葉より「あなたがいると助かる」という言葉に一番喜ぶ
  • 遅刻した相手を怒らずにはいられない（でも結局許す）

→ ESTJの恋愛詳細はこちら

ESFJ
ラブラドール・レトリバー（領事）
温かく包み込む世話焼き恋人
  • 好きな人のSNSをさかのぼって好みのリサーチをしてしまう
  • 相手が悲しそうにしていると、理由も分からないまま自分も落ち込む
  • 「嫌い」と言えなくて、我慢しすぎてある日突然爆発する
  • 二人の記念日を全部覚えていて、さりげなくお祝いする（重いと思われたくないから控えめに）
  • 「俺（私）のことを1番に考えてほしい」という気持ちがある
  • グループの場で相手がつまらなそうにしていると、自分のことより気になってしまう

→ ESFJの恋愛詳細はこちら

🌟 SP（自由派）グループのあるある

ISTP
ミニチュア・ダックスフンド（巨匠）
クールな職人気質の不器用な愛情家
  • 好意があっても「なんか一緒にいるね」「まあ悪くない」くらいしか言えない
  • デートより「一緒に作業する時間」に幸せを感じる
  • 相手が「どう思ってるの？」と聞いてくると固まる
  • 感情を言葉にするのが苦手すぎて、行動で示すしかない
  • 「もっと話してほしい」と言われても話す内容が思いつかない
  • 一人の時間が必要なのに、それを説明するのが難しい

→ ISTJの恋愛詳細はこちら

ISFP
フレンチ・ブルドッグ（冒険家）
自由を愛するアーティスト系恋人
  • 好きな人への気持ちを、手作りのもの・絵・音楽で表現しようとする
  • 感情が溢れると言葉が出なくなり、黙ってしまう
  • 「自由にしていいよ」という言葉が一番嬉しい
  • 感動したものを共有したくて、好きな人に押しつけ気味に紹介してしまう
  • 急に感情が爆発して「もう無理」となることがある（実はずっと溜め込んでいた）
  • 一緒にいる時は言葉より「この時間が好き」という雰囲気を大切にする

→ ISFPの恋愛詳細はこちら

ESTP
ビーグル（起業家）
勢いで突っ走る刺激大好き冒険家
  • 好きな人ができたら「えいやっ」で誘ってしまう（考えた後で不安になる）
  • デートは計画より「行き当たりばったり」が楽しいと思っている
  • 「将来のこと考えてる？」と聞かれると急に頭が真っ白になる
  • 感情的な話し合いより「じゃあ何か楽しいことしよ！」で解決しようとする
  • ノリで告白してしまい、後で「あれは本気だったのかな」と自分でも分からなくなる
  • 一緒にいて楽しいと感じた瞬間、本気で好きになっていることに気づく

→ ESTJの恋愛詳細はこちら

ESFP
ポメラニアン（エンターテイナー）
笑顔の太陽みたいな恋愛エンジョイ人
  • 好意が顔にそのまま出るので、みんなにバレている（本人は隠してるつもり）
  • 好きな人のそばにいるための理由を毎回考えている
  • 笑わせようとしすぎて、たまに滑る。でも本人は全力
  • 別れ際に「もう少しだけ」をなんとかして延ばそうとする
  • 「どんな子が好き？」という話題で探りを入れてしまう
  • 付き合ったら全力で幸せにしたい気持ちが強すぎて空回りすることがある
「わかる！！」ってなったら友達にも送ってみて 🐾
